# Location Tracker with Google Map Link - v1.0 


<b>Installation:</b>

# Termux

```
pkg install git

git clone https://github.com/darkstarbdx/GIPLocator1.0

cd GIPLocator1.0

chmod +x ipl.py

python3 ipl.py

```
# Linux

```
sudo apt-get update

sudo apt-get install git

sudo git clone https://github.com/darkstarbdx/GIPLocator1.0

cd GIPLocator1.0

sudo chmod +x ipl.py

sudo python3 ipl.py

```
